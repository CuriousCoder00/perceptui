## API References

| Prop      | Type                                     | Description                  | Default |
| :-------- | :--------------------------------------- | :--------------------------- | :------ |
| variant   | `"classic" \| "standard" \| "ghost"`     | Form of the Password Input   | classic |
| radius    | `"none"\|"sm"\|"md"\|"lg"\|"xl"\|"full"` | Radius of the Password Input | lg      |
| className | `string`                                 | className                    | \_      |

## Examples

### Variant

Use the `variant` prop to control the visual style of the password input.

<ComponentPreview component={
<div className="flex flex-col gap-4 p-4 bg-blue-500/15 mb-2 rounded-lg w-full">

    <PasswordInput variant="standard" />

  </div>    }>
```jsx code copy

  return (
    <div className="flex flex-col gap-4 p-4 bg-blue-500/15 mb-2 rounded-lg w-full">
    <PasswordInput variant="classic" />
    <PasswordInput variant="standard" />
    <PasswordInput variant="ghost" />
  </div>
  )
}
```
</ComponentPreview>
