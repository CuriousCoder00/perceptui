# What we offer?

- **Customizable Components**: Easily adapt the components to fit your design requirements.
- **Accessible**: Built with accessibility in mind to ensure your application is usable by everyone.
- **Well-documented**: Comprehensive documentation to help you integrate components seamlessly.
- **Templates**: Configured templates to start your project faster. Leave the configuration to us.

Percept UI is designed to work out of the box with minimal configuration with Tailwind CSS.

## CLI

We offer `@perceptui/cli`, a CLI tool to up-start your project with pre-configured templates or add components to your project.

<Callout>Checkout [`@perceptui/cli`](/docs/cli) CLI.</Callout>

## Components Library

We offer `@perceptui/ui`, a component library.
